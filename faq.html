常见问题 FAQ

1. 农民币 ($NM) 是什么？

农民币是一个 完全去中心化的社区实验项目，结合 NFT 质押与链上自动化机制构建，无预售、无融资、无操控权限，所有规则写入合约并自动执行。

2. 农民币有总量上限吗？

是的，总量固定为 9,000,000 枚，永不增发。所有代币仅通过 NFT 持有者质押产出，机制透明、无法更改。

3. 如何获得 $NM？

当前唯一获取方式为：持有 NFT 并质押至矿池。根据时间、产出比例、阶段规则等因素自动产出 $NM。

4. 有锁仓机制吗？

没有锁仓，所有产出的 $NM 自动归属用户钱包，自由转账和使用。

5. 如何查看我拥有的 $NM？

在你的以太坊钱包（如 MetaMask）中添加以下合约地址：

0x7cd04a5132fAa672314aCA0d063c90C0EA3747d4

6. NFT 有什么作用？

农民币 NFT 是项目的唯一挖矿凭证。它既是数字纪念品，也是社区身份标识。持有 NFT 可持续产出 $NM，直到挖完。

7. 项目合法吗？

本项目不募资、不承诺收益、不涉及证券。所有机制公开透明，NFT 属艺术品性质，参与者需自行判断法律适用。

8. 可以交易吗？

目前未上线任何交易所。未来是否上线 DEX 取决于社区共识，项目方不负责推动或控制交易相关行为。

9. 有奖励活动或空投吗？

没有。所有产出仅来自 NFT 质押，不再发放任何人工空投。

14. 我如何购买 NFT？

农民币 NFT 的购买不依赖中心化网页界面，而是通过链上合约直接完成。用户需自行持有足够的 USDC（链上版本） 并调用购买合约的 buy(uint256 quantity) 函数来完成购买操作。

请前往 NFT信息 页面，获取最新的购买合约地址、USDC授权方法、购买函数入口等必要信息。

15. 为什么没有一键购买界面？

农民币是一个完全去中心化的实验项目，不提供 Web2 风格的一键式购买界面。我们鼓励用户了解区块链底层逻辑，自主操作钱包与合约，这也是项目精神的一部分。

当然，若你不熟悉这些操作，可前往 NFT信息 页面查看手把手教程，或在社群中寻求帮助。

16. 矿池是否安全？会不会吞掉我的 NFT？

农民币矿池的合约代码遵循 极简、安全、可验证 的原则，机制完全写死在链上，无任何外部控制权限。

矿池本身不具备吞掉用户 NFT 的功能，用户质押后每小时只能选择两种操作：领取产出 或 取回质押，不会被强制锁仓或消失。

我们建议你先阅读 NFT 合约和矿池合约代码，或通过 区块链浏览器 验证合约函数，所有操作完全公开透明。
